LCD2S Serial LCD Display
This section is used to communicate with a LCD2S serial LCD display. The LCD2S must be configured for I2C mode (DIP switch on LCD2S set to 00 - both off).
String
Write string to LCD display. For details, see "ll" command. here
For example, the following string will write "Hello" to row 1, and "World" to row two:
_fhello_nworld
String:
Byte Array
Write bytes to LCD display. For details, see "lk" command here
For example, the following string will send a "Blink Block Cursor On" command (hex 18) to the display:
_18
