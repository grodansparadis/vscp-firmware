This is an example of how to send a compressed text file with the Modtronix TCP/IP stack. This file 
you are viewing is a compressed text file, and is sent via the HTTP server on the Modtronix TCP/IP stack. 

To included a compressed text file with your web pages all you have
to do is check the "compress" check box next to the file name in the Modtronix Network
Bootloader application when compiling the web pages. Click 
<a href="http://www.modtronix.com/soft/netloader" target="_blank">here</a> for details.
