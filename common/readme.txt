How To Use VSCP Common Code
===========================

Check the vscp-firmware.h file for a description of the methods that your firmware application 
must implement to use the vscp common code.  There is also a description here on line
https://www.vscp.org/docs/vscpfirmware/doku.php


The following files are copies from the vscp_software repository 
(https://github.com/grodansparadis/vscp_software) copied here for
developer convenience. If ever a question arise which version is the 
correct one the version in the vscp_software repository should be trusted.

canal.h
vscp.h
vscp_class.h
vscp_type.h


Functionality
=============

vscp-firmware.h/c         - Common code for VSCP firmware level I
vscp-serial.h             - Serial definitions.
vscp-fifo.h/c             - fifo for vscpEvent's
vscp-firmware-level2.h/c  - Common code for VSCP firmware level II
vscp-firmware-helper.h/c  - VSCP protocol support functionality
vscp-link-protocol.h/c
