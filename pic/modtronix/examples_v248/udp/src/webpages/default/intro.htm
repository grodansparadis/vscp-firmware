Introduction

This page is served by a HTTP Server running on a SBC65EC Single Board Computer from Modtronix Engineering. It uses the free Modtronix TCP/IP Stack (modified version of the Microchip TCP/IP Stack) and a PIC18F6621 CPU. It has 32 free I/O pins, of which 12 can be configured as Analog inputs. It has 64KBytes of Flash, 3.8K of SRAM and 64K of EEPROM for web pages. This demo applications uses less than half of the available Flash and SRAM, and less than 25% of the available EEPROM space!

The TCP/IP Stack included with the SBC65EC features:

  • Portable across Microchip PIC18 family of microcontrollers
  • Cooperative multitasking TCP/IP stack
  • Use it as part of HTTP Server or custom TCP/IP based application
  • Multiple TCP socket support
  • HTTP Server with multiple connections support
  • Pages may be stored in internal program memory or external data EEPROM
  • Support for HTML forms
  • PC based program to convert web pages into Microchip File System (MPFS)
  • Modular design to easily add new components
